Программа запускается в Matlab, желательно 2023 версии и позднее.
Стандартный запуск программы, по настройкам Matlab, файл с исходными данными input1.csv должен находится в той же папке, что и программа.
Результатом будет файл out1.csv. Также на рабочем столе Matlab будет находится весь список рассматриваемых гипотез, их веса, а также в статусе будет выведено время работы алгоритма (0.1-0.2с).
